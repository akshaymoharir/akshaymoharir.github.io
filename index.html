This fusion yields precise location and heading angle of the vehicle. Demo video shows a car was driven on 11 mile-Telegraph road and a small visualization of car moving on real time map is shown.

  • Vehicle classification using camera & LiDAR sensor fusion (YouTube video)
Using camera and LiDAR sensor fusion, measurements of objects are identified. These measurements are used to classify object using machine learning algorithm. Support Vector Machine model is trained in controlled environment which classifies objects into 4 different categories, pedestrian, cars, SUVs, trucks.
